## ⚡ Quick Commands

```bash
# Setup & Run Backend
cd backend
npm install
cp .env.example .env        # Edit with your MongoDB URI
npm run dev

# Setup & Run Frontend (in new terminal)
cd frontend
npm install
cp .env.example .env
npm run dev

# Access Application
# Frontend: http://localhost:5173
# Backend: http://localhost:5000
```
